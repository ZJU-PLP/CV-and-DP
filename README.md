## Tips
* 选择计算机视觉+深度学习方向，应该三大顶会（顶刊）等论文找一个细致方向（检测、分割或者无监督等等）跟进，充分利用论文中的开源代码，选取重点几篇论文进行复现，工程和研究能力并进！
* 深度学习框架也可以看官网的英文版教程，选择一个框架工具并要熟练掌握相关接口
